## Overview (Roman Urdu Mein)
Is lab mein humne **NumPy** library ka istemal karte huye data manipulation aur analysis seekha hai. NumPy industry mein base layer ki tarah kaam karti hai aur AI models ke backend par yahi mathematics chal rahi hoti hai.

### Humne Kya Kya Kiya (Practical Tasks):
1. **Task 12 (Arrays Creation)**: Humne 1D (vector), 2D (matrix), aur 3D (cube) arrays banaye. Is se humne seekha ke data different shapes mein kaise store hota hai.
2. **Task 13 (Indexing & Reshaping)**: Kisi bhi array se specific data nikaalna (Slicing) aur array ki shakal badalna (Reshaping) seekha.
3. **Task 14 (Maths & Broadcasting)**: Arrays par direct mathematical operations (+, -, *) kiye. Broadcasting ke zariye humne sikha ke kaise ek single number ko poore array par apply kiya jata hai.
4. **Task 15 (Statistical Functions)**: Data ka **Mean, Median, aur Standard Deviation** nikaala. Ye kisi bhi data ki "health" check karne ke liye zaroori hota hai.
5. **Task 16 (Linear Algebra)**: AI ka "Dimaag" yani **Matrix Multiplication (Dot Product)** perform kiya.
6. **Task 17 (Random Sampling)**: Simulation ke liye random numbers generate kiye.

### Final Deliverable:
Humne ek **Stock Price Visualization** chart banaya jo real-world prediction ki ek choti si simulation hai.

### Performance Comparison:
Humne proof kiya ke NumPy normal Python lists se **10x se 100x tak fast** hai jab hum heavy data par kaam karte hain.
